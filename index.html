HOW SHAPE FITTING WORKS

The model's mesh is split into clusters using a 3D grid. Each cluster's bounding box aspect ratio determines its shape: nearly cubic → box, tall thin → cylinder, very flat → flat square, tapered → cone or pyramid, irregular → sphere. Colors are sampled from vertex colors or material base colors.

AFTER EXPORTING

Go to grabvr.tools → File → Insert → .JSON File. Then File → Export → Export .level. The color palette row at the top lets you identify and fix any colors that sampled wrong.

TIPS FOR BEST RESULTS

Use GLB/GLTF for best color sampling — OBJ and STL rarely contain color data. Higher complexity = more blocks = more detail but larger files and slower Grab loading. Start at 1750 and adjust. The model is auto-scaled to fit within ~10 Grab units.
