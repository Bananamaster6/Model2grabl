readme.txt
REQUIRED GROUP NAMES

Your Grab level JSON must have levelNodeGroup nodes with name fields: L (left arm/hand), R (right arm/hand), body (torso/spine), head. With Limbs on: f1,f2,f3,f4 and l1,l2,l3,l4.

BONE MAPPING

The tool guesses which model bone matches each group using name matching (spine, pelvis, hips → body; head, neck → head; left hand, wrist → L; right hand → R; etc). You can manually override any mapping in the table above.

OUTPUT

Downloads a Grab JSON with the animation keyframes (position + rotation) applied to your character groups. Import into grabvr.tools, then export as .level and drop into your Grab levels folder.
